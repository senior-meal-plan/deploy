# Senior Meal Plan Backend

어르신들을 위한 맞춤형 식단 추천 및 관리 서비스의 백엔드 프로젝트입니다.

## 실행 방법

1.  **MySQL 설치**: 로컬 환경에 MySQL 데이터베이스를 설치해야 합니다.
2.  **환경 변수 설정**: 프로젝트 루트 디렉토리에 `.env` 파일을 생성하고, 아래와 같이 MySQL 접속 비밀번호를 입력합니다.
    ```
    RDB_PASSWORD=your_mysql_password
    ```

>  **Note:** 자세한 데이터베이스 스키마 생성 및 구동 방법에 대한 문서는 추후 업데이트될 예정입니다.

---

## 진행 상황

-   **JPA Entity 매핑**: 아래 ERD를 기반으로 프로젝트의 핵심 도메인에 대한 JPA 엔티티 설계를 완료했습니다.
